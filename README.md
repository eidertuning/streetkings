## 🇬🇧 English

## About Five Horizon Motorsport

**Five Horizon Motorsport** is a custom **FiveM racing framework project** built from the original **StreetKings** base.

This fork is focused on expanding and adapting the original framework into a motorsport-oriented experience with its own identity, custom adjustments, translations, configurations, visual improvements, gameplay changes, and future custom systems.

### Core framework features

The StreetKings base includes systems and structure related to:

- player flow and game states;
- vehicles, garages, dealerships, and progression;
- XP, economy, stats, and persistent data;
- races, freeroam activities, and mission systems;
- UI integrations, notifications, and app/resource support;
- a modular framework foundation for further expansion.

All original credit for the **StreetKings** framework, including its base concept, core systems, original structure, and original authorship, belongs to its creators.

This repository **does not claim ownership of the original StreetKings base** and does not intend to remove, replace, or hide the authorship of the original creators.

**Five Horizon Motorsport** should be understood as a **custom fork / modified version** of StreetKings.

That means:

- the **original StreetKings framework** and its base rights remain credited to its original creators;
- the **name, branding, custom edits, translations, adjustments, added systems, and project direction** of this fork belong to **Five Horizon Motorsport / this custom project**;

### Framework note

This project is being adapted as a **customized framework for a new racing environment**, with the intention of supporting:

- custom server identity;
- custom gameplay balancing;
- framework expansion;
- UI and system improvements;

- long-term custom development.
